# Pasos para correr el proyecto local

## 1 Clonar
```
git clone https://github.com/CDiazHipertexto/asegurados.git
```
## 2 entrar a la carpeta
```
cd asegurados
```

## 3 configuración del proyecto
```
npm install
```
## 4 Compila y recarga en caliente para el desarrollo
```
npm run dev
```
